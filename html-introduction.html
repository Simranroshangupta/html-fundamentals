HTML Introduction

HTML stands for HyperText Markup Language

HTML is used for developing the structure or format of a web page which is displayed in the browser.

HTML uses tags that are opened and closed using their tag names. The most important tag is the html tag, which opens first and closes last. The DOCTYPE html declaration has no closing tag and is used to specify the HTML version being used.

The head tag is used to define metadata of the web page. It can include the title, external links, CSS, and JavaScript files.

There is a difference between the head tag and headings. The head tag contains information about the page, whereas headings are used to display content on the web page. HTML provides six heading levels, from h1 to h6.

Heading Levels

h1 - Biggest Heading

h2 - Second Level Heading

h3 - Third Level Heading

h4 - Fourth Level Heading

h5 - Fifth Level Heading
h6 - Smallest Heading
