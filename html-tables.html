Learning HTML Tables

HTML tables are used to display data in rows and columns.

Tables help to organize information clearly.

Main Table Tags

table tag is used to create a table.

tr tag is used to create table rows.

th tag is used to create table headings.

td tag is used to create table data cells.

Basic Table Example

Name Status
Sriram Active
Diya Inactive

Media Tags

Media tags are used to display images, audio, and videos.

The img tag works using the src (source) attribute.

Audio Tag

The audio tag is used to play audio files on a web page.

Conclusion

Tables are an important tag in HTML.

They are used to show data clearly.

Main tags are table, tr, th, and td.

Tables are widely used in real websites.
